Rights to Post Journal Entry

This module is simple but very useful if you want to restrict the rights to post any journal entry by you
in draft state, which cannot be done using Odoo default functionality.
1. Post Journal Rights.
After installing this module, there will become visible, a Accounting Post Approval check on every
users profile.
This check will help restrict the right to post any journal entry which was created earlier in draft state.



2. After assigning the right to post any journal entry to a specific user(s), the user will be able to see the
Post button using which he will be able to Post a journal entry.
